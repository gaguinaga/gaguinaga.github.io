Learning HTML & CSS

I've been learning HTML & CSS on Khan Academy. I've tried to take lessons years ago at another online training site called Codecademy.

So the main difference between the two is that Khan Academy offers video training. Codecademy's HTML & CSS course only offers a console for users to type in. There are questions and answers on the training console which did'nt really help me understand the language.
